# Jeffrey Sedoro — Portfolio

Personal portfolio built with [Next.js](https://nextjs.org) (App Router), [Motion](https://motion.dev), and [Tailwind CSS v4](https://tailwindcss.com). Features a split-name hero with photo, text-flip role cycling, staggered text reveal, responsive mobile/desktop layouts, a Leaflet map, and a contact form.

## Tech Stack

- **Framework** — Next.js 16 (App Router)
- **Language** — TypeScript
- **Styling** — Tailwind CSS v4 + `tw-animate-css`
- **Animation** — Motion (React)
- **UI Primitives** — shadcn/ui
- **Map** — Leaflet
- **Email** — Nodemailer (Gmail SMTP)

## Color System (60-30-10)

| Layer | Light Mode | Dark Mode |
|---|---|---|
| 60% Background | `#f5f3ef` warm off-white | `#0d0d0d` near-black |
| 30% Text / Card | `#1a1a1a` / `#eae7e1` | `#f0ede8` / `#1a1a1a` |
| 10% Accent | `#7CB342` muted green | `#BEFF00` acid-green |

## Getting Started

```bash

npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.
