# Jeffrey Sedoro — Portfolio

Personal portfolio built with [Next.js](https://nextjs.org) (App Router), [Motion](https://motion.dev), and [Tailwind CSS v4](https://tailwindcss.com). Features a ripple-effect hero, interactive UI components, a Leaflet map, a scroll progress indicator with RGB lighting, and a loading screen.

## Tech Stack

- **Framework** — Next.js 16 (App Router)
- **Language** — TypeScript
- **Styling** — Tailwind CSS v4 + `tw-animate-css`
- **Animation** — Motion (React)
- **UI Primitives** — shadcn/ui + Base UI
- **Map** — Leaflet
- **Icons** — Lucide React

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Build

```bash
npm run build
```
